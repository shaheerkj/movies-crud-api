# How to run:

- `git clone https://github.com/shaheerkj/movies-crud-api`
- `cd movies-crud-api && go run ./cmd/web`

# Available Methods:

- `GET /movies` -> Returns list of all movies
- `GET /movies/{id}` -> Returns the movie with that specific id.
- `POST /movies` -> Send movie data in body and it returns the updated list of movies.
- `PUT /movies/{id}` -> Update a movie
- `DELETE /movies/id` -> Deletes a movie with that id.
